# To-Do List with Drag & Drop  
A simple and **responsive To-Do List Web App** built with **HTML, CSS, Bootstrap, JavaScript, and jQuery**. Users can **add tasks, mark them as completed, delete tasks, and reorder them using drag & drop**.  

## Features  
✔️ Add, mark as complete, and delete tasks  
✔️ Drag & drop to reorder tasks  
✔️ Smooth animations with jQuery  
✔️ Fully responsive design (Bootstrap 5)  

## Technologies Used  
- **HTML5** - Page structure  
- **CSS3 & Bootstrap 5** - Styling & layout  
- **JavaScript & jQuery** - Functionality & interactions  

## How to Use?  
1. Clone the repository:  
   git clone https://github.com/santoshhinduja-15/To-do-List.git
2. Open **index.html** in a browser.
3. Start adding and managing tasks!
